# 🤖 Workflow: AI Agent Setup

## Objetivo

Diseñar, construir y poner en producción un agente de IA autónomo con tools, memoria, RAG y prompts optimizados.

## Skills involucradas

| Fase | Skill | Propósito |
|------|-------|-----------|
| 1. Arquitectura | `ai-agents-architect` | Loops maestros (ReAct/Plan-Execute), memoria |
| 2. Tools | `agent-tool-builder` | Tools robustas y JSON schemas |
| 3. Prompting | `prompt-mastery` | Prompting avanzado (CoT, Few-Shot) |
| 4. Optimización | `prompt-engineer` | Frameworks RTF, RISEN, RODES |
| 5. RAG | `rag-expert` | Chunking semántico, reranking |
| 6. MCP | `mcp-builder` | Servidores Model Context Protocol |
| 7. Meta | `meta-skill-antigravity` | Metodología para crear nuevas skills |
| 8. Autoría | `writing-great-skills` | Skills predecibles y reutilizables |

## Flujo

```
Diseño → Tools → Prompts → RAG → MCP → Producción
```

## Outputs

- Agente funcional con tools y memoria
- Skills reutilizables documentadas
- Pipeline RAG de alta precisión
- Servidores MCP configurados
