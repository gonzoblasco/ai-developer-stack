# 📈 Playbook de Crecimiento y Marca Personal

Este playbook ofrece estrategias de marketing, copywriting y marca personal para establecer autoridad, hacer crecer audiencias y lanzar productos.

## Skills Disponibles

| Skill | Descripción | Target |
| --- | --- | --- |
| [linkedin-profile-optimizer](linkedin-profile-optimizer/SKILL.md) | Auditar y optimizar el posicionamiento del perfil de LinkedIn para autoridad y SEO | Marca Personal |
| [social-post-writer-seo](social-post-writer-seo/SKILL.md) | Generación de contenido para redes sociales optimizado para cada plataforma | Copywriting |

## Filosofía

1. **Identidad clara sobre confusión de marca**: Definí primero una "Identidad Core" clara antes de listar diversas habilidades.
2. **Métricas de Acción-Resultado**: Destacá resultados cuantificables (impacto, porcentajes, valores en dólares) en lugar de palabras de moda genéricas.
3. **Matices nativos de cada plataforma**: Adaptá el tono, la densidad de emojis y los hashtags específicamente al canal objetivo (LinkedIn vs. Instagram).
