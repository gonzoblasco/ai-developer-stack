# 📈 Growth & Personal Brand Playbook

This playbook provides marketing, copywriting, and personal branding strategies to establish authority, grow audiences, and launch products.

## Skills Available

| Skill | Description | Target |
| --- | --- | --- |
| [linkedin-profile-optimizer](linkedin-profile-optimizer/SKILL.md) | Audit and optimize LinkedIn profile positioning for authority and SEO | Professional Brand |
| [social-post-writer-seo](social-post-writer-seo/SKILL.md) | Platform-optimized social media content generation | Copywriting |

## Philosophy

1. **Clear Identity over Brand Confusion**: Define a clear "Core Identity" first before listing diverse skills.
2. **Action-Result Metrics**: Highlight quantifiable results (impact, percentages, dollar values) instead of generic buzzwords.
3. **Platform-Native Nuances**: Adapt tone, emoji density, and hashtags specifically to the target channel (LinkedIn vs. Instagram).
