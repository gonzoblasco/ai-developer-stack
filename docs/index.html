# Playbooks

Cada dominio tiene un playbook que conecta sus skills en un workflow cohesivo. No es una colección de prompts — es un sistema.

# Skills

Más de 100 skills organizadas por dominio. Cada una tiene su SKILL.md con contexto, instrucciones y ejemplos.

Cloná el stack. Construí con ventaja.

Todas las skills son gratuitas y open source. Forkealas, adaptalas, mejoralas. La comunidad hispanohablante merece documentación técnica de primer nivel.
