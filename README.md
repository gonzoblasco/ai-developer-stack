# 🚀 Antigravity Developer Stack (AGDS)

**"La magia no está en la receta, está en el cocinero. Pero una buena receta ayuda."**

> **El manual que faltaba para la Ingeniería con IA.**
> Un stack curado de 30+ patrones, prompts y workflows battle-tested diseñados para operacionalizar ingeniería de alto nivel. Desde Arquitectura de Sistemas hasta Agentes Autónomos, este es el blueprint del Desarrollador AI-First.

---

## 📚 Engineering Playbooks

No colecciones prompts. Usá **sistemas**.
Cada categoría tiene un **Engineering Playbook** detallado que te guía por el ciclo de vida de ese dominio.

### 🤖 [AI Engineering Playbook](ai-agents/README.md)

_Del Concepto al Agente Autónomo._

- **Manos:** Construí Tools robustas y JSON Schemas.
- **Cerebro:** Diseñá loops de agentes con estructuras de memoria y guardrails.
- **Contexto:** Implementá pipelines RAG de alta precisión.

### 🧠 [Strategy & Architecture Playbook](strategy-architecture/README.md)

_De la Ideación al Roadmap._

- **Ideación:** Técnicas de brainstorming estructurado.
- **Diseño:** Diagramas C4 y Technical Decision Records (ADRs).
- **Patrones:** Hexagonal, Clean Architecture y Domain-Driven Design.

### ⚙️ [Backend Engineering Playbook](backend-infra/README.md)

_Del Contrato de API al Contenedor._

- **Fundamentos:** NestJS y estructuras modulares.
- **Datos:** Normalización de esquemas, Prisma y optimización PostgreSQL.
- **DevOps:** Dockerización y readiness para producción.

### 🌐 [Full-Stack Playbook](fullstack/README.md)

_Del Vibe Coding a Producción Endurecida._

- **Salud:** Limpieza de deuda técnica generada por IA.
- **Refactoring:** Cambios incrementales seguros sin demoler.

### 🎨 [Frontend Engineering Playbook](frontend-ui/README.md)

_De los Guidelines al Factor "Wow"._

- **Estándares:** Guidelines estrictos de componentes, shadcn/ui y Tailwind CSS.
- **Arquitectura:** Next.js escalable, gestión de estado y TanStack Query.
- **UX:** Interfaces premium con micro-interacciones.

### 📏 [Code Quality Playbook](languages-standards/README.md)

_Del Tipado al Testing._

- **TypeScript:** Patrones de tipado estricto (sin `any`).
- **Principios:** Guías de Clean Code de Uncle Bob.
- **Node.js**: Arquitectura en capas, ESM y consideraciones de runtime.
- **Testing:** Mentalidad TDD, mocking con Vitest y mock factories.

### 🛠️ [Automation Playbook](tools-automation/README.md)

_Del Trabajo Manual a los Robots._

- **Low-Code:** Orquestación de workflows con n8n y webhooks de API Mercado Libre.
- **Sync:** Automatización de issues y roadmaps en Linear.
- **Browser:** Scraping resiliente, testing e2e y hooks de revisión humana.
- **Docs:** Compilación de manuales de referencia técnica de la codebase.

### 🏗️ [Product Building Playbook](product-building/README.md)

_Del Primer Prompt al MVP en Producción._

- **Stack:** Lovable Cloud + Supabase + Edge Functions.
- **Workflow:** Memoria persistente usando 8 archivos de planificación estructurados (de PRD a Tracker).
- **Filosofía:** Roadmaps escritos como planes, auditoría de vibe code e iteración incremental.

### 📈 [Growth & Personal Brand Playbook](growth/README.md)

_De Experto Técnico a Autoridad._

- **Identidad:** Posicionamiento claro y crecimiento de audiencia.
- **Copywriting:** Contenido para redes sociales nativo de cada plataforma y SEO.

### ✍️ [Skill Authoring Playbook](skill-authoring/README.md)

_Del Prompting a Skills de Agente Predecibles._

- **Estándares:** Diseñando skills de IA robustas y reutilizables.
- **Higiene:** Manejo de ventanas de contexto y divulgación progresiva.
