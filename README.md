# 🚀 The Antigravity Developer Stack (AGDS)

**"Magic is not in the recipe, it is in the cook. But a good recipe helps."**

> **The missing manual for AI Engineering.**
> A curated stack of 30+ battle-tested patterns, prompts, and workflows designed to operationalize high-level engineering. From System Architecture to Autonomous Agents, this is the blueprint for the AI-First Developer.

---

## 📚 Engineering Playbooks

Don't just collect prompts. Use **Systems**.
Each category below contains a detailed **Engineering Playbook** that guides you through the lifecycle of that domain.

### 🤖 [AI Engineering Playbook](ai-agents/README.md)

_From Concept to Autonomous Agent._

- **Hands:** Build robust Tools & JSON Schemas.
- **Brain:** Design agent loops with memory structures and guardrails.
- **Context:** Implement high-precision RAG pipelines.

### 🧠 [Strategy & Architecture Playbook](strategy-architecture/README.md)

_From Ideation to Roadmap._

- **Ideation:** Structured brainstorming techniques.
- **Design:** C4 Diagrams and Technical Decision Records (ADRs).
- **Patterns:** Hexagonal, Clean Architecture, and Domain-Driven Design.

### ⚙️ [Backend Engineering Playbook](backend-infra/README.md)

_From API Contract to Container._

- **Foundation:** NestJS and modular structures.
- **Data:** Schema normalization, Prisma, and PostgreSQL optimization.
- **DevOps:** Dockerization and production readiness.

### 🎨 [Frontend Engineering Playbook](frontend-ui/README.md)

_From Guidelines to 'Wow' Factor._

- **Standards:** Strict component guidelines, shadcn/ui, and Tailwind CSS.
- **Architecture:** Scalable Next.js, state management, and TanStack Query.
- **UX:** Premium aesthetic interfaces with micro-interactions.

### 📏 [Code Quality Playbook](languages-standards/README.md)

_From Typing to Testing._

- **TypeScript:** Strict typing patterns (No `any`).
- **Principles:** Uncle Bob's Clean Code guidelines.
- **Node.js**: Layered architecture, ESM, and runtime considerations.
- **Testing:** TDD mindsets, Vitest mocking, and mock factories.

### 🛠️ [Automation Playbook](tools-automation/README.md)

_From Manual Labor to Robots._

- **Low-Code:** Orchestrating workflows with n8n and Mercado Libre API webhooks.
- **Sync:** Automation of issues and roadmaps in Linear.
- **Browser:** Resilient scraping, e2e testing, and human review hooks.
- **Docs:** Compiling codebase technical reference manuals.

### 🏗️ [Product Building Playbook](product-building/README.md)

_From First Prompt to MVP in Production._

- **Stack:** Lovable Cloud + Supabase + Edge Functions.
- **Workflow:** Persistent memory using 8 structured planning files (PRD to Tracker).
- **Philosophy:** Plan-writing roadmaps, vibe code auditing, and incremental iteration.
